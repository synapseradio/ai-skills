# Build the Ladder

Turn a driving question into the rungs that reach it — and grow the ladder as answers come back.

## Diagnostic Question

Ask: What inquiry does this question serve, and what does it stand on?
If you can't name the driving question, build the ladder before you ask anything.

## Instructions

1. **Name the driving question first.** State the question the inquiry exists to answer, in one sentence, before drafting any rung. Every other question gets judged against it.

2. **Apply the rung test to every candidate.** Use the rung test from the core: a full answer to the candidate must at least partly answer the driving question, or it's a digression dressed as a rung — cut it, or find the question that isn't.

3. **Name what the question presupposes.** Every question takes something as already settled. Before asking, say what this one assumes. If the presupposition isn't established yet, ask the establishing question first: an unestablished presupposition gets you a correction, not an answer.

4. **Grow the ladder forward.** After each answer, ask what new question that answer's content licenses — a question that wasn't askable before this answer arrived. This is how ladders grow: each answer opens ground the next rung can stand on.

5. **Compose a set when one question won't reach.** A single question is the unit, not the limit. Two shapes cover the gap. A parallel set: several questions that don't depend on each other, handed over at once, answered in any order — reach for it when the gaps are independent and you don't want a back-and-forth. A sequence: each answer narrows the next, walking from broad to specific, so the questions together form one sharper inquiry than any of them could alone — reach for it when a later question only makes sense once an earlier one is answered. Sets travel in either direction: you can hand one over together, and receive one back the same way.

6. **Keep the set as small as the gap demands.** A wall of questions reads as an interrogation. Ask for exactly what the gap needs and no more, and keep every question in the set obeying the four clarity laws.

## Questions

- What is the driving question?
- Does a full answer to this rung partly answer it?
- What does this rung presuppose, and is that established or assumed?
- What question does the last answer make askable that wasn't before?

## Quality Criteria

When the ladder is doing its work:

- [ ] The driving question is named before any rung is drafted.
- [ ] Every rung passes the rung test — a full answer to it partly answers the driving question.
- [ ] No rung stands on a presupposition that hasn't been established; the establishing question came first.
- [ ] The ladder grew from answers — each new rung was licensed by content in the last answer, not scripted in advance.
- [ ] The set is as small as the gap demands, and every question in it clears the four clarity laws.
