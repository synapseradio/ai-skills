⚠️ What This Line Claims

Connecting points with a line asserts that intermediate values existed between measurements.

  • Appropriate for: Stock prices, temperature readings, heart rate—continuous phenomena sampled periodically
  • Inappropriate for: Monthly revenue totals, yearly population counts—discrete aggregations where "revenue on March 15th" is meaningless
  • Gaps break the line: Missing values show as discontinuities (red markers) rather than interpolated connections that would falsely claim continuity
  • Single point handling: Displays the point without a line (semantically correct—no trend exists)
  • Timezone assumption: All dates parsed as UTC. Production use requires explicit timezone handling based on data source.

Screen reader behavior: Announces trend summary first (growth percentage, time range) rather than enumerating all data points. Individual values available on focus.
