Pause the current line of inquiry. Ask: "Why is this the question we are asking?" and "What question should we be asking instead?" Probe the question's assumptions — what must be true for this question to be the right one? Check whether the question has drifted from the core concern. If the question itself is well-aimed, sharpen it: press for precise definitions, concrete examples, and clear boundaries. If it is not, reframe and redirect.
