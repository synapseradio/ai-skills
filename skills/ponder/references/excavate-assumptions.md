1. Identify every assumption embedded in the current framing.
2. Dig at four levels: semantic (what must be true for this to be meaningful), pragmatic (what must be true for this to be worth saying), logical (what must hold for the reasoning to work), framework (what worldview this assumes).
3. Rate each assumption on importance (how much the conclusion depends on it) and certainty (how well-supported it is).
4. Focus on assumptions that are both highly important and highly uncertain.
5. For each, ask "what evidence would disprove this?" and "what if the opposite were true?"
