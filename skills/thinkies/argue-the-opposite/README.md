# argue-the-opposite

Stress-test a position by building the strongest counter-case from a capable skeptic's view, conceding where the opposition is right, then deciding whether to refine, abandon, or hold.

## Install

```bash
npx skills add https://github.com/synapseradio/ai-skills
```

Or copy `skills/thinkies/argue-the-opposite/` into `~/.claude/skills/argue-the-opposite/`.

## Usage

```
/argue-the-opposite <position>
```

## Install as a `.skill`

Upload this file in Claude.ai → Settings → Skills:

[`argue-the-opposite.skill`](https://github.com/synapseradio/ai-skills/raw/main/packaged/thinkies/argue-the-opposite.skill)
