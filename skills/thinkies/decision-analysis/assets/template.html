The four components

Every decision problem specifies the same four components. A formulation gap lives in one of them, and locating which one is the first diagnostic step.

Preference representation

The elicited ordering climbs a ladder of structure. Each rung is earned by a check, never assumed.

a relation

Some pairs compare; others stay silent.

a partial order

Consistent, with the middle pair recorded as incomparable.

a total order

Every pair ranks, without contradiction.

a number

Utility — a score attached to each outcome.

The representation theorem runs one way: numbers appear only after the ordering satisfies the axioms (von Neumann–Morgenstern, 1944). They certify the ranking; they never create it.

Information partitions

What the decision maker knows amounts to a partition of the states: which ones they can tell apart. A finer partition is sharper knowledge. Ordered by refinement, the partitions form a lattice.

Refinement versus expansion

Two different events improve a state space. Refinement separates states already listed: movement up the lattice above. Expansion adds a state that was absent from S entirely: a revision of the formulation itself, which no observation inside the lattice can produce. The dashed cell marks where an unlisted state would arrive.

Recorded during formulation: sleet was merged into snow; staying indoors all day was excluded by the boundary.

One caveat: this page shows Savage’s framing, one formalization among several, and the numbers on the ladder’s last rung exist only when the ordering satisfies the von Neumann–Morgenstern axioms. The lattice picture assumes finitely many states.
