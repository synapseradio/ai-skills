1. Assume the plan has already been implemented and failed catastrophically. Write the headline.
2. Work backwards: what went wrong?
3. Generate failure modes across technical, human, process, external, and timing dimensions — without censoring.
4. Surface the hidden assumptions: what was taken for granted, what dependencies were invisible?
5. Group failure modes by theme.
6. Identify which ones the current plan does not address.
7. Design interventions for the most dangerous.
