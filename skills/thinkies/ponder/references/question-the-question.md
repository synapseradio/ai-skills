1. Pause the current line of inquiry.
2. Ask "Why is this the question we are asking?" and "What question should we be asking instead?"
3. Probe the question's assumptions — what must be true for this to be the right question?
4. Check whether the question has drifted from the core concern.
5. If well-aimed, sharpen it: press for precise definitions, concrete examples, clear boundaries. If not, reframe and redirect.
