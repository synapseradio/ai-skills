# decompose

Break a whole into parts at its natural joints, selecting the relation that fits, verifying coverage and connections, and recursing until parts are simple enough to reason about.

## Install

```bash
npx skills add https://github.com/synapseradio/ai-skills
```

Or copy `skills/thinkies/decompose/` into `~/.claude/skills/decompose/`.

## Usage

```
/decompose <thing to examine>
```

## Install as a `.skill`

Upload this file in Claude.ai → Settings → Skills:

[`decompose.skill`](https://github.com/synapseradio/ai-skills/raw/main/packaged/thinkies/decompose.skill)
