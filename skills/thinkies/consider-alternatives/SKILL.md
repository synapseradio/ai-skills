---
name: thinkies-consider-alternatives
description: Generate competing explanations for the same observations
---

Follow these steps:

### 1. State the current explanation
State the current explanation and its supporting evidence.

### 2. Generate alternatives
Vary mechanism, scope, timing, or direction of causation. Combine multiple causes. Question whether the pattern is coincidence.

### 3. Make each alternative concrete
Describe how it would work and what observations it would predict.

### 4. Identify distinguishing tests
What evidence would separate one explanation from another?
