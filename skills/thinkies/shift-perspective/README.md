# shift-perspective

Inhabit 2–4 contrasting frames — stakeholders, disciplines, time horizons, levels of expertise — to locate what each sees that the others miss and return with insight no single frame would produce.

## Install

```bash
npx skills add https://github.com/synapseradio/ai-skills
```

Or copy `skills/thinkies/shift-perspective/` into `~/.claude/skills/shift-perspective/`.

## Usage

```
/shift-perspective <situation or decision>
```

## Install as a `.skill`

Upload this file in Claude.ai → Settings → Skills:

[`shift-perspective.skill`](https://github.com/synapseradio/ai-skills/raw/main/packaged/thinkies/shift-perspective.skill)
