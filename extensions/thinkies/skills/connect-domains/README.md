# connect-domains

Import solutions from structurally similar problems in distant domains — biology, other industries, physical processes, games — then map them back and extract the underlying principle.

## Install

```bash
npx skills add https://github.com/synapseradio/ai-skills
```

Or copy `skills/thinkies/connect-domains/` into `~/.claude/skills/connect-domains/`.

## Usage

```
/connect-domains <problem>
```

## Install as a `.skill`

Upload this file in Claude.ai → Settings → Skills:

[`connect-domains.skill`](https://github.com/synapseradio/ai-skills/raw/main/packaged/thinkies/connect-domains.skill)
