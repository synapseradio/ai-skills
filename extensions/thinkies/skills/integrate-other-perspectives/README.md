# integrate-other-perspectives

Combine viewpoints into a coherent whole. Finds points of contact, addresses tensions, builds the synthesis, and tests it for coherence.

## Install

```bash
npx skills add https://github.com/synapseradio/ai-skills
```

Or copy `skills/thinkies/integrate-other-perspectives/` into `~/.claude/skills/integrate-other-perspectives/`.

## Usage

```
/integrate-other-perspectives <perspectives or viewpoints to integrate>
```

## Install as a `.skill`

Upload this file in Claude.ai → Settings → Skills:

[`integrate-other-perspectives.skill`](https://github.com/synapseradio/ai-skills/raw/main/packaged/thinkies/integrate-other-perspectives.skill)
