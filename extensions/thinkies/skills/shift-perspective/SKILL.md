---
name: shift-perspective
description: Inhabit contrasting frames to see what one viewpoint misses
---

Follow these steps:

### 1. Identify the current frame

Identify the current frame.

### 2. Choose contrasting perspectives

Choose a few — different stakeholders, disciplines, time horizons, or levels of expertise.

### 3. Inhabit each fully

What does it assume, prioritize, and protect? Use "I" statements from within each frame.

### 4. Find the blind spots

Locate what each perspective sees that the others miss.

### 5. Note the clashes

Note where perspectives genuinely clash.

### 6. Return with insight

Return with the insight that no single frame would have produced alone.
