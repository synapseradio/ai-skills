# synthesize-opposing-views

Find higher understanding through dialectic. States thesis and antithesis, examines the conflict, identifies deeper truth, and constructs a synthesis that transcends the either/or framing.

## Install

```bash
npx skills add https://github.com/synapseradio/ai-skills
```

Or copy `skills/thinkies/synthesize-opposing-views/` into `~/.claude/skills/synthesize-opposing-views/`.

## Usage

```
/synthesize-opposing-views <thesis and antithesis to synthesize>
```

## Install as a `.skill`

Upload this file in Claude.ai → Settings → Skills:

[`synthesize-opposing-views.skill`](https://github.com/synapseradio/ai-skills/raw/main/packaged/thinkies/synthesize-opposing-views.skill)
