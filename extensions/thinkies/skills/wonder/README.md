# wonder

Open the possibility space through curiosity. Notices the surprising, asks generative questions, follows interesting threads, and treats confusion as a signal about what deserves attention.

## Install

```bash
npx skills add https://github.com/synapseradio/ai-skills
```

Or copy `skills/thinkies/wonder/` into `~/.claude/skills/wonder/`.

## Usage

```
/wonder <situation or problem>
```

## Install as a `.skill`

Upload this file in Claude.ai → Settings → Skills:

[`wonder.skill`](https://github.com/synapseradio/ai-skills/raw/main/packaged/thinkies/wonder.skill)
