    C->>API: GET /resource (expired token)
    API->>Auth: Validate token
    Auth-->>API: 401 token expired
    API-->>C: 401 (with refresh hint)
    C->>Auth: POST /refresh (refresh_token)
    Auth->>DB: Lookup refresh_token
    DB-->>Auth: User + scopes
    Auth-->>C: New access_token
    C->>API: GET /resource (new token)
    API->>Auth: Validate token
    Auth-->>API: 200 OK + claims
    API-->>C: 200 OK + payload
