# invert-the-problem

Turn a problem inside out — ask how to guarantee the opposite, list every failure path, then reverse each into a protective measure — to reveal hidden structure.

## Install

```bash
npx skills add https://github.com/synapseradio/ai-skills
```

Or copy `skills/thinkies/invert-the-problem/` into `~/.claude/skills/invert-the-problem/`.

## Usage

```
/invert-the-problem <goal or problem>
```

## Install as a `.skill`

Upload this file in Claude.ai → Settings → Skills:

[`invert-the-problem.skill`](https://github.com/synapseradio/ai-skills/raw/main/packaged/thinkies/invert-the-problem.skill)
