# Pose Questions

Transform declarative prose into exploratory prose through structured questions that invite investigation.

## Diagnostic Question

Ask: Does this prose close inquiry or open it?
Where could a declaration become an invitation to explore? What questions does this raise that could be made explicit?

## Instructions

1. **Identify closures** - Detect claims that present conclusions without revealing the investigation that led there. Mark assertions that state outcomes without showing the questions that produced them. Find explanations that answer questions the reader never got to ask.

2. **Generate dimensional questions** - Create questions that guide investigation along specific dimensions rather than seeking yes-no answers. "What changes if we vary X?" or "How does this behavior shift when context changes?" open exploration. "Is X true?" closes it.

3. **Transform declarations into invitations** - Convert "X is true" into "What makes X appear true in this context?" Convert "Y causes Z" into "Under what conditions does Y produce Z, and when does that relationship break down?"

4. **Structure productive incompleteness** - Leave gaps that pull the reader forward rather than frustrate them. Incomplete prose asks "What happens next?" or "How does this connect?" Frustrating prose leaves the reader unable to proceed. The distinction is whether the gap has direction.

5. **Distinguish productive from rhetorical questions** - Productive questions genuinely invite investigation with multiple valid paths. Rhetorical questions pretend to invite but actually assert: "Isn't it obvious that X?" is a claim disguised as question.

6. **Match questions to content depth** - Surface claims invite "What's an example?" or "How does this work in practice?" Deeper claims invite "What assumptions enable this conclusion?" Complex arguments invite "What would an adversary argue?"

7. **Create question sequences** - Generate questions that build on each other. First question opens a dimension, second explores it, third tests boundaries. The sequence creates momentum rather than scattering attention.

## Questions

A short interrogation to run when prose may be closing inquiry that should stay open. The questions name what to inspect. They don't prescribe what to conclude.

- Where does the prose state an outcome without showing the question that produced it? Where could a declaration become an invitation to investigate alongside the writer?
- For each question I draft, does it open a dimension to explore, or does it only ask for a yes-no verdict?
- Where am I writing a question that pretends to invite but is actually asserting? "Isn't it obvious that..." is a claim wearing a question mark.
- Does each gap I leave for the reader pull them forward in a direction, or does it leave them unable to take a step?
- Do my questions match the depth of the claim — examples for surface claims, assumptions for deeper claims, adversary positions for contested claims?
- Where questions appear in sequence, do they build on each other and gather momentum, or do they scatter the reader's attention?

## Quality Criteria

When the questions are doing their work:

- [ ] Each declaration that warrants investigation is paired with a question the reader can follow into the inquiry.
- [ ] Each question opens a dimension to explore, with multiple valid paths through it.
- [ ] No question is asserting an answer behind a question mark.
- [ ] Gaps left for the reader carry direction. The reader can take a step from where the prose stops.
- [ ] Question depth matches claim depth — examples for surface claims, assumptions for deeper claims, adversary positions for contested claims.
- [ ] Where questions appear in sequence, each one builds on the last.
