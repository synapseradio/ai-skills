---
name: question-the-question
description: Examine whether the inquiry is aimed at the right target
---

Follow these steps:

### 1. Pause the inquiry

Pause the current line of inquiry.

### 2. Ask why this question

Ask "Why is this the question we are asking?" and "What question should we be asking instead?"

### 3. Probe its assumptions

What must be true for this to be the right question?

### 4. Check for drift

Check whether the question has drifted from the core concern.

### 5. Sharpen or reframe

If well-aimed, sharpen it: press for precise definitions, concrete examples, clear boundaries. If not, reframe and redirect.
