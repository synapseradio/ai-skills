# Redraft Workflow

Apply expression techniques in cascading sequence until prose reaches natural completion — the point where techniques stop finding substantial improvements. Each pass makes subsequent passes more effective: streamlining reveals structure that enables arc; arc reveals assumptions that invite surfacing. The cascade is multiplicative, not additive.

Contrast with the **Clarify workflow** (`@./references/workflow-clarify.md`, Clarity category only — fast, focused diagnostic) and **ad-hoc technique use** (technique access without workflow overhead, handled by the router).

## Why This Order

Pass 0 (lens) → Clarity (structure) → Integrity (honesty) → Shape (craft) → Depth (meaning) → Connection (audience).

1. **Pass 0 first** because tradition, fit, and authorial stance must be established before any technique fires. Without this lens, diagnostics default to Anglo-American conventions invisibly.
2. **Clarity second** because you cannot craft what you cannot see. Streamlining reveals actual structure. Active voice names actors. Strong language commits.
3. **Integrity third** because honest expression requires clean structure to calibrate against.
4. **Shape fourth** because craft builds on clear, honest content — arc, arrangement, rhythm, voice, and register shape what clarity and integrity prepared.
5. **Depth fifth** because opening meaning works best on well-crafted prose.
6. **Connection last** because audience calibration applies to the finished expression.

```text
Pass 0: Orient → Pass 1: Clarity → Pass 2: Integrity → Pass 3: Shape →
Pass 4: Depth → Pass 5: Connection → Converge?
     ^                                                          |
     +--------------- (if substantial changes remain) ---------+
```

## NEVER

- **NEVER** run Pass 4 on structurally disordered prose — implications extracted from unclear claims create coherent-sounding confusion, not insight.
- **NEVER** declare stabilization after one round unless all five passes found only cosmetic changes.
- **NEVER** apply Pass 3 before Pass 2 — voice on dishonest claims amplifies, not repairs, the honesty problem.
- **NEVER** strip hedges without the hedging split — distinguish epistemic calibration (preserve) from habitual cushioning (cut).
- **NEVER** skip Pass 0 — tradition, fit, and authorial stance must precede all technique application.
- **NEVER** smooth productive friction in Pass 3 — irregular rhythm, unresolved tension, and deliberate register shifts are asked about before normalized.

## Pass 0: Orient

Before any technique fires, establish the lens that modifies all downstream passes.

1. **Load tradition lens.** Identify the rhetorical tradition operating in the prose. Note how it modifies downstream passes:
   - Arabic tradition: Pass 1 structural diagnostics must accept parallel-restatement as intentional
   - French tradition: Pass 1 must accept dialectical indirection as method, not vagueness
   - Japanese tradition: Pass 2 hedging split must distinguish particles (relational care) from cushioning
   - High-context traditions: Pass 5 calibration must not maximize explicitness

2. **Assess fit.** Does the prose's form serve its purpose — does the shape match the goal? If the form-function alignment is off, downstream passes refine the wrong shape.

3. **Identify writer's authorial stance.** What is the writer's relationship to this subject? What can they legitimately claim? What are they unable to see from where they stand? Note for Pass 3 voice audit.

4. **Check prose purpose.** Is this prose resisting clarity rules deliberately? Writing-as-resistance must be recognized, not corrected.

## Pass 1: Clarity

Before any repair technique in this pass, ask: should this content exist at all? Intentional omission in high-context traditions, strategic silence as craft, and productive ambiguity that invites reader completion are not gaps to fill. If the absence serves a purpose, preserve it.

Ask: Can a reader follow — within this tradition's logic? Are actors visible? Does language commit or cushion? Do abstractions have grounding?

The structural diagnostic is tradition-relativized from Pass 0, not universal. "Can a reader follow linearly?" becomes "Does this organizational structure serve the argument's purpose within its tradition?"

Apply clarity techniques in order:

1. **Structural and content audit** (clarify): Sequence, information-flow, gaps, prerequisites. Streamline redundancy. Preserve intrinsic cognitive load — essential complexity is not a clarity problem.
2. **Voice activation** (activate): Surface hidden actors, convert passive to active. Preserve agentless constructions where culturally intentional (Arabic, Japanese passive as rhetorical choice).
3. **Language commitment** (strengthen): Eliminate hedges, weak verbs, nominalizations, filler. **Hedging split mandatory**: distinguish epistemic calibration (preserve) from habitual cushioning (cut). Cultural calibration — Japanese hedges may be relational care, not weakness; check tradition before cutting.
4. **Illustration** (illustrate): Ground abstractions in concrete instances. For category language, apply grounding moves — what witnessed detail could replace this abstraction?

## Pass 2: Integrity

Ask: Does certainty match evidence? Where does this apply vs. break down? What must be true for this reasoning to hold?

1. **Signal confidence** (signal-confidence): Match certainty language to evidence. **Hedging split mandatory** — the same distinction from Pass 1 applies here. Japanese epistemic particles (preserve) vs. habitual English cushioning (cut).
2. **Bound scope** (bound-scope): Define where claims apply and where they break down. State boundary conditions explicitly.
3. **Surface assumptions** (surface-assumptions): Make unstated premises explicit. Identify what must be true for the reasoning to hold.

## Pass 3: Shape

Ask: What tension drives this forward? Does the ordering serve the purpose? Does rhythm vary? Does voice stay consistent AND positioned? Does each word carry its weight? Does register fit purpose?

1. **Arc** (arc): Structure with tension and resolution. Create pull, not just sequence.
2. **Arrange** (arrange): Check whether the ordering of sections and arguments serves the purpose. Does the sequence build, or merely accumulate?
3. **Rhythm** (rhythm): Vary sentence length and structure. Irregular rhythm may be intentional — ask before regularizing.
4. **Voice** (voice): Establish consistent authorial presence. Does this person speak from a stated position? Generic unmarked voice defaults to dominant-culture register; make that visible. Includes authorial stance — verify the writer's relationship to the subject established in Pass 0 is carried through the prose.
5. **Register** (register): Word-level precision and emotional pitch. Eliminate AI diction patterns ("it's worth noting", "it's important to", "in conclusion"). Each word carries its weight or goes. Match emotional register to purpose.

## Pass 4: Depth

Ask: What follows from these claims that goes unsaid? Where does feedback conflate concerns? Does this prose close inquiry or open it?

**NEVER** run Pass 4 on structurally disordered prose — implications extracted from unclear claims create coherent-sounding confusion.

1. **Extract implications** (extract-implications): Surface what follows from claims that goes unstated.
2. **Dimensionalize** (dimensionalize): Reveal independent aspects of concepts that feedback conflates.
3. **Pose questions** (pose-questions): Create structured gaps that invite exploration rather than closing inquiry.

## Pass 5: Connection

Ask: What does this audience already know and what rhetorical conventions do they expect? What unfamiliar concepts appear without anchoring?

1. **Calibrate** (calibrate): Adjust depth and terminology for audience. Cultural dimension added — not just "what does the audience know?" but "what rhetorical conventions does the audience expect?" High-context audiences may find over-explanation condescending.
2. **Bridge** (bridge): Connect unfamiliar to familiar through analogy. Cross-cultural bridging as well as knowledge bridging — what concepts need tradition-aware framing?

## Convergence

After all five passes, assess whether the cascade should continue.

**Continue if:**

- Significant structural changes were made in Pass 1
- New assumptions surfaced that affect other claims
- Arc restructuring revealed new clarity opportunities
- The prose feels substantially different from where it started this round

**Stabilize if:**

- Changes are primarily cosmetic (word choice tweaks)
- Each pass finds little to improve
- The prose has reached a coherent resting state

If continuing, return to Pass 1 with refined output. Pass 0 context carries forward — tradition, fit, and authorial stance do not need re-identification unless the prose's purpose shifted during refinement. Most prose stabilizes within 2-3 full cascades.

## Usage Modes

### Full Cascade (Default)

Run all passes to natural completion. Use for significant prose where quality matters.

### Targeted Pass

When you know what the prose needs, request specific passes. Triage table:

| Symptom | Start With |
|---------|------------|
| Hard to follow, reader must re-read | Pass 1 (Clarity) |
| Claims feel overstated or underqualified | Pass 2 (Integrity) |
| Reads as assembled, not crafted | Pass 3 (Shape) |
| Feels closed or conclusory | Pass 4 (Depth) |
| Audience mismatch — too dense or too thin | Pass 5 (Connection) |
| Prose feels culturally misaligned | Pass 0 (Orient) |

When in doubt, start with Pass 1 — most prose problems are sequence and structure problems.

### Iterative Refinement

For long documents, consider section-by-section refinement:

1. Refine the introduction to establish voice and arc
2. Refine body sections for clarity and integrity
3. Refine conclusion for narrative resolution
4. Final pass on the whole for consistency

## Compounding Effects

Techniques create opportunities for other techniques:

- Streamline → reveals structure → enables arc to work
- Activate → names actors → enables confidence calibration (who claims what?)
- Strengthen → removes hedges → forces honest confidence assessment
- Arc → creates tension → makes rhythm variations meaningful
- Arrange → orders arguments → enables voice to build coherently
- Surface-assumptions → reveals hidden claims → enables confidence calibration
- Extract-implications → surfaces consequences → creates material for questions
- Dimensionalize → separates aspects → enables targeted calibration
- Illustrate (with grounding moves) → replaces categories with experience → enables register precision
- Voice (with authorial stance) → positions the writer → enables honest confidence signaling
