# Topic Evolution System

Unify tags and topics with temporal evolution tracking to make knowledge organization visible over time.

## Problem

Both threads and topics are invisible—agents don't encounter them during normal workflow. Additionally, topics lack a temporal dimension (they're snapshots, not timelines).

## Key Capabilities

- **Unified tags → topics**: Eliminate the tags/topics distinction
- **Temporal tracking**: `first_seen`, `last_seen`, mention history with timestamps
- **Provenance**: Track who said what when about each topic
- **Surface at workflow entry points**: Topics appear at session start
