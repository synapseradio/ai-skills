# Discourse Resume & Extend

Extend discourse sessions with resume and extend capabilities, allowing multi-perspective dialogues to persist across sessions.

## Problem

Discourse sessions are ephemeral—when interrupted, the conversation is lost. Participants can be re-instantiated but without memory of what was said.

## Key Capabilities

- **Resume**: Continue a dialogue with the same participants, preserving full context
- **Extend**: Add new perspectives to an existing dialogue
- **Session history**: Track rounds completed across multiple sessions
